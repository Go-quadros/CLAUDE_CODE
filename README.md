# CLAUDE_CODE

Repositório de experimentos e aprendizado com Claude Code.

## O que é este projeto?

Este repositório foi criado para explorar as capacidades do Claude Code — a CLI oficial da Anthropic para desenvolvimento assistido por IA.

## Como começar

1. Instale o [Claude Code](https://claude.ai/code)
2. Clone este repositório
3. Explore os exemplos na pasta do projeto
